# Welcome to TorrentPier

TorrentPier is a bull-powered BitTorrent tracker engine written in PHP. It's an open-source software that allows you to run your own private or public torrent community with integrated forum functionality.

## Key features

- **High performance**: Optimized for handling thousands of peers
- **Forum integration**: Built-in forum for your torrent community
- **Modern PHP**: Requires PHP 8.4+ with modern practices
- **Template engine**: Twig templates with backward compatibility
- **Database flexibility**: MySQL/MariaDB/Percona support
- **Extensible**: Clean architecture designed for modifications

## Project status

TorrentPier is in active development. Version 3.0 brings major improvements including mod support and Twig template engine. See the [blog](/blog) for development updates.
